# 🚀 Projeto de Portfólio: QA Manual - Sauce Demo (Swag Labs)

Este repositório contém a documentação completa de testes manuais realizados no site [Sauce Demo](https://www.saucedemo.com). O objetivo é demonstrar habilidades em planejamento de testes, execução de cenários, geração de evidências e reporte de bugs.

## 🛠️ O que foi desenvolvido:

- **Plano de Teste:** Definição de escopo, ferramentas e estratégia. ([Ver Documento](./documentacao/PlanoDeTeste.md))
- **Casos de Teste:** Escrita e execução de 5 cenários principais (Login, Carrinho e Regras de Negócio). ([Ver Tabela](./casos-de-teste/CasosDeTeste.md))
- **Evidências:** Capturas de tela comprovando a execução de cada teste. ([Ver Pasta](./evidencias/))
- **Bug Report (Simulação):** Demonstração técnica de como reportar uma falha crítica de segurança. ([Ver Bug](./bugs/SIMULACAO-BUG001-falha-bloqueio.md))

## 🧪 Tecnologias e Ferramentas:
- **Metodologia:** Testes de Caixa Preta (Manual).
- **Gestão:** GitHub (Pastas, Markdown e Issues).
- **Ambiente:** Google Chrome / Windows.
